For anyone who wants to save some time while looking for specific tile types:
h = roads only
m = standard roadmap
p = terrain
r = somehow altered roadmap
s = satellite only
t = terrain only
y = hybrid

also, we can add more link:
http://mt1.google.cn/vt/lyrs=h&hl=en-US&gl=cn&x={x}&y={y}&z={z}
